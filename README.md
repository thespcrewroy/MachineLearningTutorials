# DS/ML/AI Tutorials

## Data Analytics
* **Google Analytics:** web analytics platform that tracks and reports website traffic, user behavior, and marketing performance
* **Meta Pixel:** tracking tool from Meta for measuring actions and optimizing ads on Facebook and Instagram
* **Tiktok Pixel:** monitors user activity on a website to measure ad performance and optimize TikTok advertising campaigns
* **Snapchat Pixel:** helps measure conversions, optimize ads, and build audiences for Snapchat advertising campaigns

## Business Analytics/Intelligence, Management Info Systems, & Supply Chain
* **PowerBI:** business analytics platform by Microsoft that transforms data into interactive dashboards and visual reports
* **Powerpoint:** presentation software application by Microsoft used to create slide-based visual presentations
    * **Think Cell:** a PowerPoint add-in that streamlines the creation of professional charts, graphs, and data-driven presentations
* **Celonis:** a process mining and execution management platform that analyzes business processes to optimize them
* **Excel:** a spreadsheet application by Microsoft used for organizing, analyzing, calculating, and visualizing data
